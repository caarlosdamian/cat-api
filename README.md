# Cat api Application 🚀

Practice application for the consumption of the  "THE CAT API" of postman

## Installation 📋

Please run this commands to install all dependencies 

```bash
npm install
```

## Usage 🔧

```bash
npm start
```
or
```bash
yarn start
```
